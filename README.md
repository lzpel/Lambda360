## アピールポイント

### AI⇒コード⇒レンダリング⇒AIの改善ループを回せる

### 多数のシミュレーターを備えている

- レンダリングもシミュレーターの一つとみなします。（可視光があたったときに写像を出力するシミュレーター）
- ほかのシミュレーターとして流体解析、応力解析なども考えています

### wasm化してブラウザでも操作できる

これは初期の売り上げを建てるために必要です

ブラウザーにgoogleの検索窓のような枠があり

「歯車のCADを書いて」

と書いたら歯車のCADを出力するLambda360のコードとその実行結果が表示されます。
